StudyWithPomodoro

A website designed to help users study using the Pomodoro technique. Users will be able to customize the timer and use two types of widgets: a to-do list widget and a task completion widget with percentage tracking. They will also be able to track their study activity by creating an account.

The main purpose of this project is to learn the fundamentals of JavaScript, Node.js, Next.js, PostgreSQL, and software architecture in general.
